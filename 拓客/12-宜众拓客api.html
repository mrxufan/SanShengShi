宜众拓客Api

资源引用地址获取方法：

1. 在【资源管理】页面选择对应资源项目，点击进入【查看】

2. 获取资源引用地址

3. 在资源引用地址面后可按要求传入业务参数

请求数据

名称 类型 描述
id 整型 资源id，后台可查看
bizContent 字符串 请求参数字符串（建议加密）

用户兑换资源回调参数

名称 类型 描述
bizContent 字符串 请求时传入参数，原样传回
reciveContent Json字符串 资源相关内容
apiType 字符串 1资源 2活动
appName 字符串 应用名称
activeName 字符串 资源或活动名称
changeMobile 字符串 兑换手机号
reciveMobile 字符串（资源为实物有值） 收货人手机号码
reciveName 字符串（资源为实物有值） 收货人姓名
reciveAddress 字符串（资源为实物有值） 收货人地址
receiveTime 字符串（资源为实物有值）
yyyy-MM-dd HH:mm:ss		 领取时间
changeTime 字符串（资源为实物有值）
yyyy-MM-dd HH:mm:ss		 兑换时间

回调数据解密后

{
    "bizContent": "89456",
    "reciveContent": {
        "apiType": "2",
        "appName": "flappy-bird",
        "activeName": “十九大”,
        "changeMobile": "17796655539",
        "reciveMobile": "",
        "reciveName": "",
        "reciveAddress": "",
        "receiveTime": "2017-10-01 00:12:17",
        "changeTime": "2017-10-18 13:10:22"
    }
}

请求方式：post请求，请求地址在后台api设置

加密方式：Des 加密
